### 介绍

基于开源项目 [Pichi](https://github.com/pichi-router/pichi) 开放的 API 进行开发的 Web 界面控制器

主要为 React16 + Router4 + Redux 实现

该项目只完成部分功能，等待 Pichi 项目更新

### 运行
请先自行设置代理
http://localhost:[该项目运行端口号]/api/* 映射为 http://localhost:[pichi项目运行端口号]/

在项目根目录执行

`yarn run start`

或者 

`npm run start`
